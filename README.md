# 明日方舟剧情对话
通过脚本从wiki(prts)获取明日方舟大部分剧情对白

# 创建环境
安装需要的Python库：
requests

# 如何使用
先运行obtain_titles.py从Wiki(prts)获取到大部分的标题

然后运行main.py获取到剧情对白，获取到的文件将以每个关卡行动前/后单独保存到txts目录下

# 新增获取干员部分资料
运行obtain_operators.py获取干员列表到 干员列表.txt

然后运行operators.py获取每个干员的档案资料，具体词条可以选择
