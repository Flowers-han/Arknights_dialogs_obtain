# 明日方舟剧情对话
通过脚本从wiki(prts)获取明日方舟大部分剧情对白

# 创建环境
安装需要的Python库：
requests
beautifulsoup4

# 如何使用
先运行obtain_titles.py从Wiki(prts)获取到大部分的标题

然后运行main.py获取到剧情对白，获取到的文件将以每个关卡行动前/后单独保存到txts目录下
